Decisions locked in (with the why)

  1. Project = SEPIA (Systematic Evaluation of Pigmentation In Archaic hominins); a QC-first re-analysis continuing PAINT.
  2. Build strategy — report/integrate on hg38; genotypes physically stay hg19; bridge by a verified rsID↔︎hg19↔︎hg38 map. A genotype is build-independent, so this is lossless relabeling — no re-alignment. The genome-wide sanity PCA stays hg19 (a control). Genome-wide hg38 (B2b) is optional/deferred. (This reconciled an earlier hg38-monolithic drift and a mid-session hg19 over-correction.)
  3. Site build: migrated workflowr → Quarto (reconciled onto main); auto-deploy paused (workflow_dispatch) until the R-analysis pages are rendered (they need data / will be regenerated by SEPIA).
  4. Trust: rebuild from raw/published sources with QC gates; do not trust Lily's derived data.
  5. Sample subset (papers/SAMPLE_INCLUSION.md): high-cov (Altai/Den5, Vindija 33.19, Denisova 3, Chagyrskaya 8, Denisova 25) → published snpAD VCFs, no BAM; Mez1 → its VCF; low-cov Les Cottés, Denisova 11, Goyet → BAMs (ANGSD GLs + mapDamage); excluded: El Sidrón (exome), Vindija 87 (= Vi 33.19), Mez2, Spy, HST, Scladina.
  6. Modern reference: complete the 166-SGDP merge → 159 usable (7 were truncated local copies, dropped). Sanity 1 PASSED — recovers textbook global structure.
  7. Shared genomes = full canonical genomes (reusable), + cluster→GitHub push via SSH + SAML-SSO.
  8. Merge architecture lesson: NFS punishes random seeks — stream-thin (bcftools view -T) in parallel, don't merge -R over scattered regions.
